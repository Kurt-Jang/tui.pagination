페이지 네이션

- 전체 아이템 : 500

- 기타기본 옵션 적용

- 페이지 이동 전에 발생되는 beforeMove와, 페이지 이동후에 발생하는 afterMove를 등록한다

        실행코드

        var pagination = new ne.Component.Pagination({
            "itemCount": 500
        }, $('.paginate1'));

        pagination.attach('beforeMove', function(eventData) {
            console.log('beforeMove', eventData);
        }).attach('afterMove', function(eventData) {
            console.log('afterMove', eventData);
        })
