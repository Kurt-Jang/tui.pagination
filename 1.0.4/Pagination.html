    The class name is granted first page link item
    pageTemplate String <optional>
    		 "<a href='#'>{=page}</a>"

    The markup template to show page item such as 1, 2, 3, .. {=page} will be changed each page number.
    currentPageTemplate String <optional>
    		 "<strong>{=page}</strong>"

    The markup template for current page {=page} will be changed current page number.
